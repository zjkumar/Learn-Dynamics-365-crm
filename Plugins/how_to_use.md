# Registration Tool

- Download and install Microsoft.CrmSdk.XrmTooling.PluginRegistrationTool from nuget package in your visual studio

- open the PluginRegistrationTool application.

- Register a new assembly.

- Browse for the new assembly (project_name.dll file)

- choose sandbox and database options and register.

- Now your plugin is visible on the tool.

- Register new step


# Considerations

- The timeout is 2 minutes.
